    • HTML & CSS

    HTML and CSS are building blocks of a website. HTML is used for the structure of a website and, CSS designs the website to make it look good. HTML stands for HyperText Markup language. CSS stands for Cascading Style Sheet.

    Tags

    HTML is made up of tags. Tags wrap content in HTML pages to interpret it accordingly and, different tags interpret text, audio, images, and videos. Each tag has a starting and closing tag. For example, heading tags are used to define the headings of a page and six different levels: h1, h2, h3, h4, h5, and h6. Tags have attributes that are words used to control the behaviour of the tag.

    Inline and Block Elements

    A block element always starts at a new line and takes up all the width. While the inline element stays at the same line. The Block element has a top and, bottom margin and the inline element does not.

    Important Tags in HTML

    The most important tags in HTML are HTML, head, title and body. The div tag is one of the most important tags in HTML and it is used to wrap other tags and divide content on the web page. The p tag is used to wrap a paragraph.

    Semantic HTML

    Semantic elements are used to describe the meaning of the page to the browser and screen reader. The header is the top content on the web page and the footer is the bottom content. The nav is used for the navigation bar of the page.

    CSS(Cascading Style Sheet)

    CSS uses many rules to style the HTML tags. There are selectors, property, and values. For example,

    Body { background-color: lightblue; }

    The body is the selector, the background colour is the property and light blue is the value. A tag can also be selected by addressing their class or id. For example,

    HTML

    <div class="container-class" id="container-id"></div>

    CSS

    .container-class {
      background-color: black;
    }
    #container-id {
      background-color: grey;
    }
